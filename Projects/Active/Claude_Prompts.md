# Claude Prompts for Content Creation
**Quick reference for daily content workflow**

---

## Tweet Drafting

### From Voice Memo / Raw Thoughts
```
Turn this into 2 tweets using my structures.

Pillar focus: [Health / Entrepreneurship / Martial Arts / AI-Systems]

My voice: direct, no fluff, grounded in experience. No emojis unless it adds meaning.

Raw input:
[paste your voice memo transcript or notes]
```

### From Specific Structure
```
Write a tweet using the [structure name] format.

Topic: [your topic]

My voice: direct, philosophical but practical, no victim framing.

Structure:
[paste the structure template from strategy doc]
```

### Quick Tweet Polish
```
Polish this tweet. Keep my voice — direct, no fluff. Don't add emojis or hashtags.

Draft:
[your rough tweet]
```

## Reply Drafting

### Standard Reply
```
Help me reply to this tweet.

My voice: direct, add genuine insight, no flattery.

Goals:
- Add value (share experience, different angle, or genuine question)
- Sound like a peer, not a fan
- Keep it short (1-3 sentences max)

Tweet:
[paste the tweet you're replying to]

My initial thought: [optional - your rough take]
```

### Reply to Big Account (5K+ followers)
```
Help me write a reply that adds value and might get noticed.

My voice: confident, insightful, not try-hard.

Their tweet:
[paste tweet]

What I want to add: [your angle/experience]

Make it punchy — they see hundreds of replies.
```

### Reply When You Disagree
```
Help me respectfully disagree with this tweet.

My voice: direct but not combative. Confident in my position.

Their tweet:
[paste tweet]

Why I disagree: [your reasoning]

Keep it short. State my view clearly without attacking them.
```

## Newsletter Drafting

### Full Newsletter Draft
```
Draft a newsletter on this topic using my framework.

Topic: [topic]
Pillar: [Health / Entrepreneurship / Martial Arts / AI-Systems]

Framework:
1. Hook (100-150 words) — specific moment/story, make it visceral
2. Story (300-400 words) — expand the moment, include dialogue/internal thoughts
3. Lesson (300-400 words) — what this taught me, connect to larger principle
4. Application (200-300 words) — practical takeaway for reader
5. CTA (50-100 words) — invite discussion, ask question

My voice: direct, philosophical but practical, no victim framing.

Raw material:
[paste your notes, voice memo transcript, or rough ideas]
```

### Newsletter Hook Only
```
Write 3 different hooks for a newsletter about [topic].

Each hook should:
- Start with a specific moment (not abstract)
- Be visceral — what did I feel/see/hear?
- Make the reader want to know what happened next

My situation: [brief context]
```

### Newsletter Edit Pass
```
Edit this newsletter draft.

Check for:
- Weak openings (cut throat-clearing)
- Passive voice (make active)
- Fluff words (delete them)
- Flow between sections
- Clear takeaway

Keep my voice — don't make it generic.

Draft:
[paste draft]
```

## DM Drafting

### Genuine Connection DM
```
Help me write a short DM to someone whose content I appreciated.

Their content: [what they posted that resonated]
Why it hit: [your genuine connection to it]

Rules:
- No pitch, no ask
- 2-3 sentences max
- Sound like a human, not a networker
```

### Follow-Up DM (After They Replied)
```
They replied to my DM. Help me continue the conversation naturally.

Original DM: [what I said]
Their reply: [what they said]

Keep it conversational. Don't force it. If there's nothing to say, tell me.
```

## Thread Drafting (Phase 2)

### Newsletter → Thread
```
Turn this newsletter into a Twitter thread (8-12 tweets).

Rules:
- First tweet must hook hard (no "Thread:" or "1/")
- Each tweet stands alone but flows into next
- End with clear takeaway or question
- No fluff — every tweet earns its place

Newsletter:
[paste newsletter]
```

## Idea Generation

### Weekly Brainstorm
```
Generate 10 tweet ideas for this week.

Pillar focus: [Health / Entrepreneurship / Martial Arts / AI-Systems]

What happened this week:
[brief notes on experiences, insights, wins, struggles]

Mix of structures:
- 3 observation/insight tweets
- 3 lesson/principle tweets
- 2 list tweets
- 2 question/engagement tweets
```

What happened:
[describe the resistance moment]

[the internal dialogue]

[action taken or not taken]

Turn this into a tweet using the "Resistance Log" structure.

## My Voice Guidelines (Reference)

When I say "my voice," I mean:
- **Direct** — say it plainly, no hedging
- **Grounded** — speak from experience, not theory
- **Philosophical but practical** — depth with takeaway
- **Confident** — no apologizing for my views
- **No victim framing** — I chose this path, I own it
- **Minimal emojis** — only if they add meaning
- **No hashtags** — they look desperate
- **Short sentences** — punch, don't ramble
