## Setup (5 Steps, 10 Minutes)

### 1. Move This Folder
```bash
# Move the PersonalOS folder to your home directory
mv PersonalOS ~/ 
```

### 2. Run Setup
```bash
cd ~/PersonalOS
./setup.sh
```

### 3. Activate It
```bash
source ~/.zshrc   # Mac users
# or
source ~/.bashrc  # Linux users
```

### 4. Test It
```bash
today
```

You should see: "âœ… Today.md generated successfully!"

### 5. Open It
```bash
open ~/PersonalOS/Tasks/Today.md
```

**That's it. You're done.**
